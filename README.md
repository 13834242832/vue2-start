# vuejs-start

> vue2 从头再来

## vue2起手式

> 逐步搭建vue2全家桶

- [x] npm i vue-cli -g
- [x] vue init webpack-simple-2.0 vuejs-start
- [x] cd vuejs-start && npm install
- [x] npm run dev  (http://localhost:8080)
- [x] npm i vue-router@next
